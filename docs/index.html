Black Box Test Plan

Test Case ID Test Title / Feature Preconditions Test Steps (in plain language) Expected Results Actual Results Status (Pass/Fail)
TC-001 Create Account (Success) User is on the "Sign Up" screen. 1. Enter a new, valid email. 2. Enter a valid password. 3. Click the "Sign Up" button. The user is logged in and sees the main dashboard. A confirmation message appears.
TC-002 Create Account (Failure) An account with the email already exists. 1. Enter the existing email. 2. Enter any password. 3. Click the "Sign Up" button. An error message appears: "Email is already used" or "Please log in."
TC-003 Login (Success) An account with valid email & password exists. User is logged out. 1. Go to the Login screen. 2. Enter valid email. 3. Enter the correct password. 4. Click "Login." The user is logged in and sees the main dashboard.
TC-004 Login (Failure) Account exists, user logged out. 1. Go to Login. 2. Enter valid email. 3. Enter wrong password. 4. Click "Login." Error message appears: "Invalid email or password." User is not logged in.
TC-005 Add Manual Expense User is logged in and on dashboard. 1. Click "Add Expense." 2. Enter Amount: $15.50. 3. Select Category: Food. 4. Enter today's date. 5. Click "Save." The new $15.50 expense appears in "Recent Transactions."
TC-006 Verify Dashboard Update Follow steps from TC-005. 1. View "Spending by Category" pie chart. 2. View "Monthly Budget Remaining." Food slice increases; Budget Remaining decreases by $15.50.
TC-007 Edit Expense User has existing "$10.00 - Coffee" expense. 1. Find the expense. 2. Tap "Edit." 3. Change Amount to $12.00. 4. Click "Save." The list shows "$12.00 - Coffee." Dashboard totals update.
TC-008 Delete Expense User has an existing "$12.00 - Coffee" expense. 1. Find the expense. 2. Tap "Delete." 3. Confirm "Yes." The expense is removed and totals update.
TC-009 View Spending Report User has Food and Transport expenses. 1. Go to Reports. 2. Select correct date range. 3. View pie chart. The pie chart shows Food and Transport in correct proportions.
TC-010 Over-Budget Notification User has $50 Social budget, $40 spent. 1. Add new expense: $15.00 in Social. 2. Click "Save." User receives alert: "You are over your Social budget."
TC-011 Export CSV Report User is on Reports tab. Click "Export as CSV." A CSV file is downloaded.
